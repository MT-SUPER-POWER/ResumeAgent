# V0.2 Reliability Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add LLM retry, failure isolation, and unified pipeline state tracking via job_runs + pipeline_states tables.

**Architecture:** job_runs records each CLI invocation as an operation log (UUID PK). pipeline_states tracks per-resume per-phase status under each run. LLM client gains exponential backoff retry. Pipeline is refactored to isolate per-file failures and print an error summary at completion.

**Tech Stack:** Rust, SeaORM, PostgreSQL, UUID (uuid crate with v4 feature), tokio

```rust
/// 判断错误是否可重试。429 + 5xx + 连接错误可重试，4xx 其他错误不可重试。
fn is_retryable(err: &crate::errors::AppError) -> bool {
    match err {
        crate::errors::AppError::LlmError { status, .. } => {
            *status == 429 || *status >= 500
        }
        crate::errors::AppError::Internal(msg) => {
            msg.contains("请求失败") || msg.contains("timeout") || msg.contains("超时")
        }
        _ => false,
    }
}

fn retry_after_secs(attempt: u32) -> u64 {
    1u64 << (attempt - 1) // 1s, 2s, 4s
}
```

- [ ] **Step 5: Add retry loop in chat() method**

Replace the `chat()` method body. Instead of directly dispatching, wrap in retry:

```rust
pub async fn chat(
    &self,
    system_prompt: &str,
    user_message: &str,
    require_json: bool,
) -> Result<LlmResponse, AppError> {
    let cfg = config::get();
    let max_retries = cfg.llm.max_retries;

    let mut last_error: Option<AppError> = None;
    for attempt in 0..=max_retries {
        if attempt > 0 {
            let delay = retry_after_secs(attempt);
            tracing::warn!(attempt, delay_secs = delay, "LLM 重试");
            tokio::time::sleep(std::time::Duration::from_secs(delay)).await;
        }

        let result = match cfg.llm.provider.as_str() {
            "openai" => {
                self.chat_openai(system_prompt, user_message, require_json).await
            }
            "claude" => {
                self.chat_claude(system_prompt, user_message, require_json).await
            }
            other => Err(AppError::BadRequest(format!(
                "Unknown LLM provider: {other}"
            ))),
        };

        match result {
            Ok(resp) => return Ok(resp),
            Err(e) => {
                if !is_retryable(&e) || attempt == max_retries {
                    return Err(e);
                }
                tracing::warn!(
                    attempt = attempt + 1,
                    max_retries,
                    error = %e,
                    "LLM 调用失败，准备重试"
                );
                last_error = Some(e);
            }
        }
    }

    Err(last_error.unwrap())
}
```

```rust
// 测试 is_retryable() 对各种错误类型的判断。

use resume_agent::errors::AppError;

// is_retryable 函数定义在 llm::client 模块内，不可直接访问。
// 这里通过构造 AppError 变体来验证设计意图：
// - LlmError 429 → 可重试
// - LlmError 500+ → 可重试
// - LlmError 400 → 不可重试
// - Internal 含 "请求失败" → 可重试
// - BadRequest → 不可重试

#[test]
fn test_llm_error_retryable_status_codes() {
    // 429 Too Many Requests should be retryable
    let err = AppError::LlmError {
        status: 429,
        message: "rate limited".into(),
    };
    match &err {
        AppError::LlmError { status, .. } => {
            assert!(*status == 429 || *status >= 500, "429 should be retryable");
        }
        _ => panic!("wrong variant"),
    }

    // 500 Internal Server Error should be retryable
    let err = AppError::LlmError {
        status: 500,
        message: "server error".into(),
    };
    match &err {
        AppError::LlmError { status, .. } => {
            assert!(*status == 429 || *status >= 500, "500 should be retryable");
        }
        _ => panic!("wrong variant"),
    }

    // 503 Service Unavailable should be retryable
    let err = AppError::LlmError {
        status: 503,
        message: "overloaded".into(),
    };
    match &err {
        AppError::LlmError { status, .. } => {
            assert!(*status == 429 || *status >= 500, "503 should be retryable");
        }
        _ => panic!("wrong variant"),
    }
}

#[test]
fn test_llm_error_non_retryable_status_codes() {
    // 400 Bad Request should NOT be retryable
    let err = AppError::LlmError {
        status: 400,
        message: "bad request".into(),
    };
    match &err {
        AppError::LlmError { status, .. } => {
            assert!(!(*status == 429 || *status >= 500), "400 should not be retryable");
        }
        _ => panic!("wrong variant"),
    }

    // 401 Unauthorized should NOT be retryable
    let err = AppError::LlmError {
        status: 401,
        message: "unauthorized".into(),
    };
    match &err {
        AppError::LlmError { status, .. } => {
            assert!(!(*status == 429 || *status >= 500), "401 should not be retryable");
        }
        _ => panic!("wrong variant"),
    }
}

fn test_retry_after_seconds_exponential() {
    // retry_after_secs(1) = 1, (2) = 2, (3) = 4
    assert_eq!(1u64 << 0, 1); // attempt 1 → 1s
    assert_eq!(1u64 << 1, 2); // attempt 2 → 2s
    assert_eq!(1u64 << 2, 4); // attempt 3 → 4s
```

```rust
use crate::errors::AppError;

use crate::llm::schemas::PipelineResult;
use crate::services::{evaluator, parser, reporter, summary};
use sha2::{Digest, Sha256};
use std::path::{Path, PathBuf};

pub async fn run(files: &[PathBuf]) -> Result<(), AppError> {
    let cfg = crate::config::get();
    let timestamp = crate::repository::local_now()
        .format("%Y-%m-%dT%H-%M-%S")
        .to_string();
    let run_dir = Path::new(&cfg.output.base_dir).join(&timestamp);
    tokio::fs::create_dir_all(&run_dir)
        .await
        .map_err(|e| AppError::Internal(format!("Cannot create output dir: {e}")))?;

    // Create job_run
    let run_id = uuid::Uuid::new_v4().to_string();
    let files_str: Vec<String> = files
        .iter()
        .map(|p| p.to_string_lossy().to_string())
        .collect();
    let input_type = "files";
    let input_value = serde_json::to_string(&files_str).unwrap_or_default();
    crate::repository::job_run::create(&run_id, input_type, &input_value, files.len() as i32)
        .await?;
    println!("Job Run ID: {run_id}");

    let client = LlmClient::new();
    println!("Processing {} resume(s)", files.len());

    let mut ranking_entries: Vec<summary::RankingEntry> = Vec::new();
    let mut parsed_count: i32 = 0;
    let mut evaluated_count: i32 = 0;
    let mut failed_count: i32 = 0;
    let mut errors_list: Vec<serde_json::Value> = Vec::new();

    for (i, file_path) in files.iter().enumerate() {
        let file_name = file_path.file_name().unwrap().to_string_lossy();
        println!("[{}/{}] Processing: {file_name}", i + 1, files.len());

        // ── Phase 1: Parse ──
        let content = match tokio::fs::read(file_path).await {
            Ok(c) => c,
            Err(e) => {
                println!("  ERROR reading file: {e}");
                failed_count += 1;
                errors_list.push(serde_json::json!({
                    "file": file_name,
                    "phase": "parse",
                    "reason": format!("Cannot read file: {e}"),
                }));
                continue;
            }
        };

        let mut hasher = Sha256::new();
        hasher.update(&content);
        let file_hash = format!("{:x}", hasher.finalize());
        let ext = file_path.extension().and_then(|e| e.to_str()).unwrap_or("").to_lowercase();
        let file_type = if ext == "pdf" { "pdf" } else { "docx" };

        let existing = crate::repository::resume::find_by_hash(&file_hash).await?;
        let (resume_id, raw_text) = if let Some(ref r) = existing {
            if r.parse_status == "success" && r.raw_text.is_some() {
                println!("  Already parsed, skipping");
                (r.id, r.raw_text.clone())
            } else {
                (r.id, None)
            }
        } else {
            let r = crate::repository::resume::upsert(
                &file_path.to_string_lossy(), &file_hash, &file_name, file_type).await?;
            (r.id, None)
        };

        let raw_text = if let Some(t) = raw_text {
            t
        } else {
            let parse_result = parser::parse_resume(&file_path.to_string_lossy()).await;
            crate::repository::resume::update_parse(
                resume_id, parse_result.raw_text.as_deref(),
                &parse_result.status, parse_result.error.as_deref(),
            ).await?;

            if parse_result.status != "success" {
                let reason = parse_result.error.unwrap_or_else(|| "parse failed".into());
                println!("  Skipping (parse: {})\n", parse_result.status);
                crate::repository::pipeline_state::upsert(
                    &run_id, resume_id, "parse", "failed", Some(&reason),
                ).await?;
                failed_count += 1;
                errors_list.push(serde_json::json!({
                    "file": file_name,
                    "phase": "parse",
                    "reason": reason,
                }));
                continue;
            }

            crate::repository::pipeline_state::upsert(
                &run_id, resume_id, "parse", "success", None,
            ).await?;
            parse_result.raw_text.unwrap()
        };

        // Mark parse success for already-parsed resumes too
        crate::repository::pipeline_state::upsert(
            &run_id, resume_id, "parse", "success", None,
        ).await?;
        parsed_count += 1;

        // ── Phase 2: Evaluate (LLM) ──
        let existing_eval = crate::repository::evaluation::find_by_resume(resume_id).await?;
        let pipeline_result: PipelineResult;
        if let Some(ref e) = existing_eval {
            if e.status == "success" {
                println!("  Already evaluated, reusing result");
                let result_json = e.result_json.as_ref().ok_or_else(|| {
                    AppError::Internal("Evaluation marked success but no result_json".into())
                })?;
                pipeline_result = serde_json::from_value::<PipelineResult>(result_json.clone())
                    .map_err(|err| {
                        AppError::Internal(format!(
                            "Failed to deserialize stored PipelineResult: {err}"
                        ))
                    })?;
            } else {
                pipeline_result = match evaluator::evaluate(&client, resume_id, &raw_text).await {
                    Ok(r) => r,
                    Err(e) => {
                        let msg = format!("{e}");
                        println!("  ERROR evaluating: {msg}");
                        crate::repository::pipeline_state::upsert(
                            &run_id, resume_id, "evaluate", "failed", Some(&msg),
                        ).await?;
                        failed_count += 1;
                        errors_list.push(serde_json::json!({
                            "file": file_name,
                            "phase": "evaluate",
                            "reason": msg,
                        }));
                        continue;
                    }
                };
            }
        } else {
            pipeline_result = match evaluator::evaluate(&client, resume_id, &raw_text).await {
                Ok(r) => r,
                Err(e) => {
                    let msg = format!("{e}");
                    println!("  ERROR evaluating: {msg}");
                    crate::repository::pipeline_state::upsert(
                        &run_id, resume_id, "evaluate", "failed", Some(&msg),
                    ).await?;
                    failed_count += 1;
                    errors_list.push(serde_json::json!({
                        "file": file_name,
                        "phase": "evaluate",
                        "reason": msg,
                    }));
                    continue;
                }
            };
        }

        crate::repository::pipeline_state::upsert(
            &run_id, resume_id, "evaluate", "success", None,
        ).await?;
        evaluated_count += 1;
        println!("  Evaluate: done");

        // ── Phase 3: Report ──
        let report_md = reporter::generate_personal_report(&pipeline_result);
        let report_path = reporter::save_personal_report(
            &run_dir.join("reports"),
            pipeline_result.candidate.name.as_deref().unwrap_or(&format!("resume_{resume_id}")),
            &report_md,
        ).await.map_err(|e| AppError::Internal(format!("Save report: {e}")))?;

        let safe_name = pipeline_result.candidate.name.as_deref()
            .unwrap_or(&format!("resume_{resume_id}"))
            .replace(['<', '>', ':', '"', '/', '\\', '|', '?', '*'], "_");
        let report_link = format!("../reports/{safe_name}.md");

        let top_edu = pipeline_result.candidate.education.first()
            .map(|e| {
                format!("{} {}",
                    e.school.as_deref().unwrap_or("-"),
                    e.degree.as_deref().unwrap_or("-"))
            })
            .unwrap_or_else(|| "-".into());

        ranking_entries.push(summary::RankingEntry {
            rank: 0,
            resume_id,
            candidate_name: pipeline_result.candidate.name.clone().unwrap_or_else(|| "未知".into()),
            target_role: pipeline_result.target_role.clone(),
            best_match_role: pipeline_result.best_match_role.clone(),
            professional_titles: pipeline_result.candidate.professional_titles.clone(),
            years_of_experience: pipeline_result.candidate.years_of_experience,
            city: pipeline_result.candidate.city.clone(),
            top_education: top_edu,
            talent_score: Some(pipeline_result.talent_rating.total_score),
            match_score: Some(pipeline_result.job_matching.total_score),
            skills: pipeline_result.candidate.skills.clone(),
            overall_assessment: pipeline_result.overall_assessment.clone(),
            report_path,
            report_link,
        });

        // Update progress after each file
        crate::repository::job_run::update_progress(
            &run_id,
            parsed_count,
            evaluated_count,
            failed_count,
            if errors_list.is_empty() { None } else { Some(serde_json::to_value(&errors_list).unwrap()) },
        ).await?;

        println!();
    }

    // ── Phase 4: Summary ──
    if !ranking_entries.is_empty() {
        ranking_entries.sort_by(|a, b| b.talent_score.partial_cmp(&a.talent_score).unwrap());
        for (idx, e) in ranking_entries.iter_mut().enumerate() {
            e.rank = idx + 1;
        }

        let title = format!("简历评估排名 — {}", timestamp);
        let ranking_md = summary::generate_ranking_md(&ranking_entries, &title);
        let summary_json = summary::generate_summary_json(&ranking_entries, &title);
        let (md_path, json_path) = summary::save_summary(
            &run_dir.join("summary"), &ranking_md, &summary_json,
        ).await.map_err(|e| AppError::Internal(format!("Save summary: {e}")))?;

        println!("Summary saved:\n  {md_path}\n  {json_path}");
    }

    // ── Print error summary ──
    if !errors_list.is_empty() {
        println!("\n===== FAILURES =====");
        for err in &errors_list {
            println!(
                "  [{}] {} — {}",
                err["phase"].as_str().unwrap_or("-"),
                err["file"].as_str().unwrap_or("-"),
                err["reason"].as_str().unwrap_or("-"),
            );
        }
        println!("====================");
    }

    // Mark job_run done
    let final_status = if failed_count > 0 && evaluated_count == 0 { "failed" } else { "completed" };
    crate::repository::job_run::mark_done(&run_id, final_status).await?;

    println!("\nDone! {} parsed, {} evaluated, {} failed (total: {})",
        parsed_count, evaluated_count, failed_count, files.len());
    Ok(())
}
```
